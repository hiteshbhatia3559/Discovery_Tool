Problem Statement
```
1. Get the IP and MAC on a provided subnet and run nmap for all addresses found.
2. Output the results to a CSV file
```

Instructions
``` 
0. Install Python 3.5
0.5. Open 'cmd', type 'ipconfig' and note your IP address
1. Install nmap and make sure nmap is added to PATH
2. Download this project as a ZIP file and extract to ./Project/
3. Once done, open a terminal/command prompt at this location and type 'python netdiscover.py'
4. Once prompted, input your IP address
5. Wait as the program finds IPs and port scans them
6. Your result will be in Results.csv
```
